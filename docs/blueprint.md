# **App Name**: NBKR BusTracker

## Core Features:

- Authentication: User authentication system with role-based access control (student, faculty, admin, driver).
- Real-time Tracking: Real-time bus tracking displayed on an interactive map with route visualization and ETA updates.
- Dashboard: Role-specific dashboards with bus assignment, location sharing, route display, and emergency alerts.
- Location Confirmation: "Are you in a bus?" confirmation button and "Share Location" activation.

## Style Guidelines:

- Primary color: Light blue (#20B2AA) for a calm and trustworthy feel, fitting for a college environment.
- Background color: Very light blue (#FFFFFF), a desaturated version of the primary color to provide a clean backdrop.
- Accent color: Blue (#4682B4), for interactive elements, providing a clear visual cue to users.
- Body and headline font: 'Inter', a sans-serif font, chosen for its clean, modern, and readable design, suitable for both headings and body text.
- Use clear and recognizable icons from a professional library (e.g., Feather Icons) to represent bus routes, stops, and other relevant information.
- Mobile-first responsive design, ensuring a consistent experience across all devices.
- Use subtle CSS animations for smooth transitions and feedback to user interactions (e.g., button clicks, location updates).
